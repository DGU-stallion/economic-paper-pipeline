## 一、变量定义

| 变量 | 角色 | 度量 |
|------|------|------|
| scr | Y | 供应链韧性（缩尾后） |
| dt | D | 数字化转型指数 |
| process | 中介1 | 流程优化度 |
| innov | 中介2 | 企业创新能力 |
| info_share | 中介3 | 信息共享水平 |
| lev | 控制 | 财务杠杆 |
| indep | 控制 | 独董比例 |
| size | 控制 | 企业规模 (ln) |
| age | 控制 | 企业年龄 (ln) |
| roa | 控制 | 盈利能力 |

## 二、基准回归：渐进控制模型

| 模型 | 设定 | 目的 |
|------|------|------|
| M1 | `scr = β₀ + β₁·dt + ε` | 零控基准 |
| M2 | M1 + lev + indep | 财务治理控制 |
| M3 | M2 + size + age | 企业特征控制 |
| M4 | M3 + roa | 全变量 |
| M5 | M4 + i.year | +年份FE |
| M6 | M5 + firm cluster SE | 完整模型 |

全部使用企业层面聚类稳健标准误。

## 三、中介效应检验

**三步法（Baron-Kenny）**：

```
Step 1: scr = c·dt + Controls + ε        (总效应)
Step 2: M = a·dt + Controls + ε          (D→中介)
Step 3: scr = c'·dt + b·M + Controls + ε (直接+间接)
```

| 路径 | 中介变量 | 假设 |
|------|---------|------|
| 路径1 | process | H2a：流程优化度 |
| 路径2 | innov | H2b：企业创新能力 |
| 路径3 | info_share | H2c：信息共享水平 |

辅助：Sobel 检验 + Bootstrap 置信区间。

## 四、异质性分析

| 维度 | 分组方式 | 假设 |
|------|---------|------|
| 企业规模 | 中位数分组（大 vs 小） | 大型企业 DT 效果更强 |
| 行业 | 制造业 vs 非制造业 | 制造业韧性提升更显著 |
| 地区 | 东部 vs 中西部 | 东部数字化基础设施更好 |

## 五、稳健性检验

| 检验 | 方法 |
|------|------|
| 替换 Y | scr 用另一种度量（如有） |
| 替换 D | dt 分位数分组 / 滞后一期 |
| 内生性 | IV：行业-地区均值 dt 作为工具变量 / 滞后 D |
| 样本剔除 | 剔除 2020 年（疫情异常冲击） |

## 六、门槛效应（非线性）

```
scr = β₀ + β₁·dt·I(q ≤ γ) + β₂·dt·I(q > γ) + Controls + ε
```

门槛变量 q = dt 自身（是否存在边际递减），交叉验证门槛数量。
