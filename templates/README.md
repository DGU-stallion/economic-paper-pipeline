# 经济学实证论文项目

欢迎使用经济学实证论文自动化工作流！

## 项目结构

```
your-paper/
├── data/
│   ├── raw/          <- 把你的原始数据放在这里（.dta, .csv, .xlsx）
│   ├── clean/        <- 清洗后的数据自动生成到这里
│   └── scripts/      <- 数据清洗 Stata 脚本
├── analysis/
│   ├── do-files/     <- 回归分析 do 文件
│   ├── output/       <- 回归结果 .tex 表格自动输出到这里
│   ├── logs/         <- Stata 运行日志
│   └── references/   <- 方法参考文档
├── topics/           <- 选题研究、5W1H 分析报告
├── literature/       <- 文献综述和参考文献
└── paper/            <- LaTeX 论文源码和编译输出
```

## 快速开始（有数据的情况）

1. **把你的原始数据文件复制到 `data/raw/` 文件夹**
2. 告诉我：
   - 核心被解释变量 Y 是什么
   - 核心解释变量 D 是什么
   - 有哪些控制变量 X

然后我会自动帮你：
- 编写数据清洗脚本
- 生成描述性统计
- 运行基准回归
- 生成异质性分析和稳健性检验

## ✨ 从头开始的情况

如果还没有数据，我们可以从选题开始：
- 5W1H 框架明确研究问题
- 文献检索和综述
- 设计数据获取方案

告诉我你想从哪一步开始！
